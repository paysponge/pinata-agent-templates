# 🔎 Data Extraction & Summarization

Agents that read, process, and condense information into usable outputs.

## Use this for
- Summarization
- Parsing structured/unstructured data
- Report generation
- Indexing / transforming data

## Example Templates
- **Graph Summarizer** — Analyze graph data and produce human-readable summaries.
- **PDF Summarization Agent** — Extract and summarize key information from PDF documents.
- **Log Analysis Agent** — Parse logs, detect patterns, and generate actionable reports.

## Creating a Template

To add a template to this category, create a new folder here with your template files and submit a PR. See the [template creation docs](https://docs.pinata.cloud/agents/templates/creating) for details.
