# 🧩 Orchestration & Multi-Agent Systems

Agents that coordinate other agents, tools, or workflows.

## Use this for
- Multi-step pipelines
- Agent coordination
- Tool routing / decision engines
- Complex workflow composition

## Example Templates
- **Research Pipeline Orchestrator** — Chain together search, extraction, and summarization agents into a research workflow.
- **Multi-Agent Task Runner** — Distribute subtasks across specialized agents and aggregate results.
- **Tool Selection / Routing Agent** — Dynamically pick the right tool or agent for each step in a workflow.

## Creating a Template

To add a template to this category, create a new folder here with your template files and submit a PR. See the [template creation docs](https://docs.pinata.cloud/agents/templates/creating) for details.
