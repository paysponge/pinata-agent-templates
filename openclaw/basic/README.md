# 🧱 Basic

Starter templates that provide a vanilla foundation for building any kind of agent.

## Use this for
- Getting started with Pinata Agents
- Learning the manifest and workspace structure
- Scaffolding a new agent from scratch
- Understanding the bootstrap flow

## Example Templates
- **[Useful Assistant](./useful-assistant)** — A generic base template with a documented manifest, workspace conventions, and a Vite + React starter app.

## Creating a Template

To add a template to this category, create a new folder here with your template files and submit a PR. See the [template creation docs](https://docs.pinata.cloud/agents/templates/creating) for details.
