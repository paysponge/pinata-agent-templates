# 💬 Interaction & Interfaces

Agents that directly interact with users or act as an interface layer.

## Use this for
- Chatbots
- Assistants
- Conversational workflows
- User-facing AI tools

## Example Templates
- **Slack Assistant** — A conversational agent that integrates with Slack for team Q&A and task management.
- **Customer Support Bot** — Handle inbound support requests, answer FAQs, and escalate when needed.

## Creating a Template

To add a template to this category, create a new folder here with your template files and submit a PR. See the [template creation docs](https://docs.pinata.cloud/agents/templates/creating) for details.
