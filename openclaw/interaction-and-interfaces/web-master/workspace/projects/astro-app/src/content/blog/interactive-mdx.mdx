## How It Works

This post is an `.mdx` file instead of `.md`. MDX lets you import and use any Astro component directly in markdown.

To create an interactive post:

1. Create a `.mdx` file in `src/content/blog/`
2. Add imports after the frontmatter
3. Use components inline with your markdown

The components render on the server. Client-side scripts hydrate when needed for interactivity.
