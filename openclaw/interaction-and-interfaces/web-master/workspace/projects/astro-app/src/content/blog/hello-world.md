Welcome to the blog. This site is built with [Astro](https://astro.build) in SSR mode, backed by a SQLite database for the waitlist, and powered by markdown content collections for the blog.

## What's Here

- **Landing page** with a waitlist signup form
- **This blog** — new posts are just markdown files
- **SQLite backend** — lightweight, local, no external dependencies

## How Posts Work

Every `.md` file dropped into the content directory becomes a blog post automatically. Add frontmatter with a title, description, and date, rebuild, and it's live.

No CMS. No database for content. Just files.
