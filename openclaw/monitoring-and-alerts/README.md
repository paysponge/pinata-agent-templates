# 🛰️ Monitoring & Alerts

Agents that observe systems, detect changes, and notify or trigger actions.

## Use this for
- Price / market monitoring
- Wallet or transaction tracking
- System health checks
- Event detection and alerting

## Example Templates
- **Financial Tracker** — Monitor prices, portfolios, or market conditions and alert on thresholds.
- **API Uptime Monitor** — Continuously check API endpoints and notify on downtime or degraded performance.
- **Activity Alert Agent** — Watch for specific on-chain or off-chain activity and send notifications.

## Creating a Template

To add a template to this category, create a new folder here with your template files and submit a PR. See the [template creation docs](https://docs.pinata.cloud/agents/templates/creating) for details.
